# 时间能量伙伴

一个手机优先的个人时间与能量管理网页。它把柳比歇夫时间统计法、语音计划、心情记录、灵感池、固定习惯、日历时间块和备份中心放在同一个地方。

## 日常使用不需要 Codex

Codex 只是帮你生成和修改代码。真正使用时，它就是一个普通网页 App：

- 你可以把它部署到 GitHub Pages、Netlify、Vercel、Cloudflare Pages。
- 你和伴侣只需要打开网页链接。
- 在 iPhone Safari 里可以把它“添加到主屏幕”。
- 记录保存在各自手机浏览器本地，不会互相覆盖。

## 本地电脑测试

```bash
pnpm install
pnpm run dev
```

然后在电脑浏览器打开终端显示的网址，通常是：

```text
http://127.0.0.1:5173/
```

如果没有 pnpm，也可以用：

```bash
npm install
npm run dev
```

## 手机同 Wi-Fi 测试

电脑和手机需要连同一个 Wi-Fi。

1. 在电脑上运行：

```bash
pnpm run dev:mobile
```

2. 查看电脑的局域网 IP。

Windows 可以运行：

```powershell
ipconfig
```

找到类似 `192.168.x.x` 的 IPv4 地址。

3. 在 iPhone Safari 打开：

```text
http://你的电脑IP:5173/
```

例如：

```text
http://192.168.1.23:5173/
```

如果打不开，通常是 Windows 防火墙没有允许 Node/Vite 通过。可以先允许专用网络访问，或者直接部署到 GitHub Pages 后用正式网址测试。

## 放到 iPhone 主屏幕

1. 用 iPhone Safari 打开网页。
2. 点分享按钮。
3. 选择“添加到主屏幕”。
4. 以后像打开 App 一样打开它。

## 连接 iPhone 操作按钮

先确保你有一个能打开的网页地址，可以是本地同 Wi-Fi 地址，也可以是 GitHub Pages 正式地址。

1. 打开“快捷指令”App。
2. 新建快捷指令。
3. 添加操作“打开 URL”。
4. 填入这个工具的网址。
5. 打开 iPhone“设置”。
6. 进入“操作按钮”。
7. 选择“快捷指令”，再选择刚刚创建的快捷指令。

之后按一下操作按钮，就会快速打开这个工具。

## 大文件放百度云盘

当前 App 适合直接保存文字灵感和小附件。大文件建议这样处理：

1. 把大文件上传到百度云盘。
2. 生成分享链接。
3. 如果有提取码，把提取码也复制下来。
4. 在灵感卡里记录：

```text
文件：访谈录音 2026-07-02
位置：百度云盘
链接：https://...
提取码：xxxx
说明：这段录音对应今天早晨散步灵感
```

这样完整备份里保存的是“文件索引”，大文件本体留在百度云盘，不会把备份文件撑得很大。

## 备份中心

底部进入“我的”，可以看到“备份中心”：

- 完整备份：导出一个 JSON 文件，包含设置、时间记录、灵感文字和小附件。
- 导入数据：用之前导出的 JSON 文件恢复。
- 导出已选灵感：先在首页“灵感池”勾选卡片，再批量导出为文本。
- 附件库：可以添加小文件，单独保存回本地，也会进入完整备份。

每张灵感卡片也有：

- 复制：复制成可分享的文字卡片。
- 导出：保存为 `.txt` 文件。
- 分享：调用手机系统分享面板。如果浏览器不支持，会退回到复制。

建议每周做一次完整备份，保存到 iCloud Drive、电脑文件夹或私人网盘。

## 开源部署

1. 在 GitHub 新建一个公开仓库。
2. 把这个文件夹里的代码提交上去。
3. 在仓库设置里开启 GitHub Pages，Source 选择 GitHub Actions。
4. 把生成的网址发给对方。

更详细的发布步骤见 [DEPLOY.md](./DEPLOY.md)。

## 如果未来不用 Codex

仍然可以继续使用和维护：

```bash
pnpm install
pnpm run dev
pnpm run build
```

`pnpm run build` 会生成 `dist/` 文件夹。这个文件夹是纯静态网页，可以放到任何静态网站服务器。

真正需要长期保存的是这个项目文件夹，不是 Codex 对话。

## 隐私

当前版本没有后端，也不会上传数据。记录保存在本机浏览器中。导出的 JSON 文件请自己保管。

注意：GitHub Pages 只放网页代码，不要把个人备份文件提交到公开仓库。
